Passo a passo - O que fazer quando criar um arquivo

1º: Criar a página HTML

2º: Criar o escopo CSS e fazer as referências das tags

3º: Nomear os containers - Ex.: cabeçalho, corpo, rodapé

4º: Colocar os containers no CSS e adicionar o display flex

5º: Criar os filhos do container e estilizar

6º: Conferir se as medidas estão fluídas

7º: Conferir se as imagens estão responsivas

8º: Definir os breakpoints


Layout fluído

Utilizar %

EM: medida relativa baseada no pai

REM: medida relativa baseada na raíz que tem o tamanho inicial de 16px e é conform o tamanho da tag HTML

VW e VH: medida relativa baseada no viewport - 1 VW ou VH vale 1% do viewport do dispositivo

  VW: largura do viewport (área de exibição do conteúdo)

  VH: altura do viewport do dispositivo que está acessando a página


Imagem responsiva

Imagem fica no tamanho normal em tela grande e no tamanho máximo quando diminui em telas menores

Utilizar: max-widtgh: ;


Breakpoints

Baseada pelo viewport

Utilização:


480px: @media (max-width: 480px)

481-768px: @media (min-width: 481px) and (max-width: 768px)

769-1279px: @media (min-width: 769px) and (max-width: 1279px)

1280px: @media (min-width: 1280px)
