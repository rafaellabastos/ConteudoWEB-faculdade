Nomes de cores

Preenchimento de fundo

  • Background-color: define a cor de fundo
  • Background-image: define a imagem de fundo
  • Background-repeat: repeat-x: repete na horizontal
  • Background-repeat: repeat-y: repete na vertical
  • Background-repeat: repeat: epete nas duas posições
  • Background-repeat: no-repeat: não se repete

  • Background-attachment: fixed: imagem permanece na mesma posição da página
  • Background-attachment: scroll: imagem se move com conteúdo quando viola a página

  • Background-position: posição da imagem

Fontes

font-family

  • Define a fonte do texto
  • font-family: arial, calibri, serif, sans-serif, monospace

fonts-google

  • Utilizar fonte do https://fonts.google.com/
  • Colocar no topo do CSS: @import url ('link da fonte')
  • font-family: nome da fonte

font-size

  • Define o tamanho da fonte

font-weight

  • font-weight: normal - para que o texto apareça normal
  • font-weight: bold - para que o texto apareça em negrito

font-style

  • font-style: normal - estilo normal
  • font-style: italic - estilo itálico

font-variant

  • font-variant: small-caps - caps um pouco abaixo da caixa alta

Textos

text-align

  • Para alinhar o texto
  • left
  • right
  • center
  • justify

text-transform

  • text-transform: uppercase - texto em maiúsculo
  • text-transform: lowercase - texto em minúsculo
  • text-transform: capitalize - primeiras letras maiúsculas

text-decoration

  • text-decoration: none - remove decorações aplicadas

  • text-decoration: underline - adiciona linhas abaixo do texto

  • text-decoration: overline - adiciona linhas acima do texto

  • text-decoration: line-through - adiciona linhas no meio do texto

  • text-decoration: underline overline - adiciona linha em cima e embaixo do texto

text-shadow

  • Cria sombras nas letras do texto
  • text-shadow: horizontal, vertical, distorção, cor

Outras configurações

  • line-height: espaçamento entre linhas
  • letter-spacing: espaçamento entre letras
  • word-spacing: espaçamento entre palavras
  • text-ident: recuo da primeira linha
