Formulários

Tag form

  • Seção do documento com controles interativos para o usuário passar informações.
  • Dentro da tag form há alguns atributos.

Atributo name

  • Identificador do formulário

Atributo action

  • Ação que o formulário terá assim que o botão submit for pressionado

Atributo method

  • Forma de envio dos dados do formulário - get ou post

Tag input

  • Define vários tipos de entrada de dados e controle
  • Dentro da tag input há alguns atributos

Atributo name

  • Especifica o nome do campo - será utilizado para recuperar o valor inserido pelo usuário

Atributo type

  • Especifica o tipo de dado de entrada que o campo receberá
    • button: insere um botão
    • checkbox: campo de seleção múltipla
      • 1 2
    • date: inserção de datas a partir de um calendário
    • datetime-local: inserção de data e hora
    • email: caixa de e-mail
    • file: campo para busca e anexo de documentos
    • month: entrada/seleção de mês e ano
    • number: entrada de números - define faixa de valores com min e max; define intervalo com step
    • password: formato para senhas
    • radio: campo de seleção única com várias opções


    • ranger: renderiza uma barra para selecionar uma escala de valores - possui min, max e step
    • reset: limpa as informações digitadas no formulário
    • search: campo de busca
    • submit: envia os dados digitados para o action
    • tel: número de telefone
    • text: digitação de dados
    • time: inserção de hora e minuto
    • url: caixa de endereço web com endereço completo
    • week: entrada/seleção da semana

Tag label

  • Serve para rotular o campo, ou seja, dizer para o usuário para que serve aquele campo
  • Para relacionar um label com um input, o for no label e o id no input devem ser os mesmos




Tag fielset

  • Elementos do formulário ficam agrupados e permitem uma maior organização dos dados no desenho do formulário
Exemplo:

Tag legend

  • Recebe um texto que permite identificar o grupo de informações que serão inseridas ali

    • Dados de cadastro




Tag textarea

  • Caixa de texto de múltiplas linhas

Tag selection/ option

  • Permite a escolha de uma opção em uma lista acessada por um botão do tipo dropdown

Outras tags

required: torna um campo do formulário obrigatório

autofocus: dá foco automático a um determinado campo assim que o navegador carregar a página

autocomplete: habilita ou desabilitada a sugestão de autocomplete do navegador

maxlenght: limita a quantidade de caracteres em um campo de formulário
