localStorage

Existem duas possibilidades principais para o armazenamento de dados no navegador com ReactJS: localStorage e sessionStorage

A diferença entre eles é que os dados da sessionStorage é apagado todas vez que o navegador é fechado/ recarregado

Métodos do objeto localStorage

  • setItem()
  • getItem()
  • removeItem()
  • clear()

Utilizando localStorage

Para salvar uma informação

  • Utilizar o método setItem() - ele deve receber uma chave e um valor

  • localStorage.setItem("chave", "valor");

Para recuperar uma informação

  • Utilizar o método getItem() - ele deve receber uma chave

  • localStorage.getItem("chave");

Para remover um item

  • Utilizar o método removeItem() - ele deve receber uma chave

  • localStorage.removeItem("chave");

Para limpar tudo

  • Utilizar o método clear()

  • localStorage.clear();

Salvando matrizes no localStorage

var ourArray = [1, 2, 3, 4, 5]; --- matriz com números

localStorage.setItem("ourarraykey", JSON.stringify(ourArray)); --- salvando no localStorage a matriz

➤ Para a matriz ser salva tem que estar convertida em string


Recuperando dados no localStorage

➤ Tem que primeiro converter a string de volta para uma matriz

var storedArray = localStorage.getItem("ourarraykey");

ourArray = JSON.parse(storedArray);
