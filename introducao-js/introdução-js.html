Variáveis e tipos de dados

  • Faz diferenciação entre letras maiúsculas e minúsculas
  • Primeira palavra a letra é minúscula e o resto é maiúsculo

  • Exemplo:
    • let nome; - variável com valor indefinido
    • let disciplina = null; - variável vazia
    • let texto = "~~" - variável com valor definido

  • Tipos de variáveis:
    • int - let idade = 18
    • float - let valor = 5.35
    • String - let nome = "Rafaella"
    • booleanos - let casado = "true"
    • array - let linguagens = ["java", "C#", "Python"]
